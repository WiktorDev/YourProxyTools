# YourProxyTools
Easy tools for bungeecord and its forks

### Functions

* Custom commands
* Soon more

###### config.yml

```yaml
commands:
  lobby:
    - '[say] #ff9999&ltest'
    - '[send] lobby'
  discord:
    - '[say] Discord: https://dc.yourcraft.pl'
```

###### Build project

```bash
mvn package
```
